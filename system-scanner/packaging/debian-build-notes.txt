==============================================================================
  System Scanner — .deb Packaging Notes
==============================================================================

This document describes how to build a .deb package for System Scanner
on Ubuntu / Debian systems.

── Prerequisites ────────────────────────────────────────────────────────

  sudo apt install dpkg-dev debhelper fakeroot

── Directory Structure for .deb ─────────────────────────────────────────

  system-scanner-deb/
  ├── DEBIAN/
  │   ├── control
  │   ├── postinst
  │   └── prerm
  └── opt/
      └── system-scanner/
          ├── app.py
          ├── run.sh
          ├── requirements.txt
          ├── scanner/
          │   ├── __init__.py
          │   ├── constants.py
          │   ├── network_utils.py
          │   ├── scanner_engine.py
          │   ├── verifier.py
          │   ├── history_manager.py
          │   └── exporter.py
          ├── assets/
          │   ├── icon.png
          │   └── style.qss
          └── data/
  └── usr/
      └── share/
          ├── applications/
          │   └── system-scanner.desktop
          └── icons/
              └── hicolor/
                  └── 256x256/
                      └── apps/
                          └── system-scanner.png

── DEBIAN/control ───────────────────────────────────────────────────────

  Package: system-scanner
  Version: 1.0.0
  Section: net
  Priority: optional
  Architecture: amd64
  Depends: python3 (>= 3.7), python3-pip, python3-venv, libgl1-mesa-glx, libegl1, libxkbcommon0, libdbus-1-3
  Maintainer: System Scanner Project <noreply@example.com>
  Description: Network scanning and port detection utility
   System Scanner is a desktop network utility for detecting open TCP
   ports on target hosts. It provides a clean PySide6 GUI with scan
   history, export, and port verification features.
  Homepage: https://github.com/example/system-scanner

── DEBIAN/postinst ──────────────────────────────────────────────────────

  #!/bin/bash
  set -e

  # Create venv and install dependencies
  cd /opt/system-scanner
  python3 -m venv venv
  ./venv/bin/pip install --upgrade pip --quiet
  ./venv/bin/pip install -r requirements.txt --quiet

  # Ensure data dir exists
  mkdir -p /opt/system-scanner/data
  chmod 755 /opt/system-scanner/data

  # Make run script executable
  chmod +x /opt/system-scanner/run.sh

  # Update desktop database
  if command -v update-desktop-database >/dev/null 2>&1; then
      update-desktop-database /usr/share/applications || true
  fi

  echo "System Scanner installed successfully."

── DEBIAN/prerm ─────────────────────────────────────────────────────────

  #!/bin/bash
  set -e

  # Remove venv
  rm -rf /opt/system-scanner/venv
  rm -rf /opt/system-scanner/data

  echo "System Scanner cleaned up."

── Build Command ────────────────────────────────────────────────────────

  1. Create the directory structure above with all files in place.

  2. Set permissions:
       chmod 755 system-scanner-deb/DEBIAN/postinst
       chmod 755 system-scanner-deb/DEBIAN/prerm

  3. Build the .deb:
       dpkg-deb --build system-scanner-deb system-scanner_1.0.0_amd64.deb

  4. Install the .deb:
       sudo dpkg -i system-scanner_1.0.0_amd64.deb
       sudo apt-get install -f   # fix any missing dependencies

  5. Uninstall:
       sudo dpkg -r system-scanner

── Notes ────────────────────────────────────────────────────────────────

  - The postinst script creates a venv inside /opt/system-scanner/ so
    PySide6 is installed per-app, not system-wide.
  - Data is stored in /opt/system-scanner/data/. For multi-user setups,
    you may want to move history to ~/.local/share/system-scanner/.
  - Tested on Ubuntu 21.04 with kernel 5.11.
